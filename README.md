# 🔐 Password Strength Checker

## 🚀 My First Cybersecurity Project

I built a Password Strength Checker using Python that evaluates password security based on factors like length, complexity, and special characters.

Through this project, I learned how weak passwords can be identified and improved using simple logic.

I’m excited to keep learning and building more in cybersecurity!

---

## 💡 Project Description

This is a simple Python project I built to check the strength of a password.

It analyzes the password based on different factors like length, uppercase and lowercase letters, numbers, and special characters.

---

## 💡 Features
- Checks password length
- Detects uppercase and lowercase letters
- Identifies numbers and special characters
- Classifies password as Weak, Medium, or Strong

---

## 🛠️ Tech Used
- Python
- Regular Expressions (re module)

---

## 🚀 How to Run
1. Download the file  
2. Open terminal  
3. Run:
   python password_checker.py
